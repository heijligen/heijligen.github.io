openSUSE bootstick for APU2

To install openSUSE on APU2 board you've tomodify the installer. I've used live-fat-stick to create an bootable usb stick. Then mounted the stick and modified btto/syslinux/syslinux.cfg
...
append initrd=initrd-openSUSE-Leap-42.3 isofrom=/dev/disk/by-uuid/[...]/openSUSE-[Version].iso isofrom_device=/dev/disk/by-uuid/[...] isofrom_system=/openSUSE-[Version].iso loader=syslinux splash=silent quiet showopts textmode=1 console=ttyS0,115200n81 usessh=1 sshpassword="my_password" ...
